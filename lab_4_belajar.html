Belajar List

Macam-Macam List :

  1. Ordered List

    2. Ordered list adalah jenis list berurutan yang ditampilkan dengan menggunakan angka atau nomor atau huruf.

  2. Unordered List

    3. Unordered list adalah jenis list yang tidak berurutan yang ditampilkan dengan menggunakan simbol.

  3. Description List

    4. Description List adalah list yang berisi deksripsi atau penjelasan dari sesuatu.

  4. List di dalam List (Nested List)

    5. List juga dapat dibuat di dalam list, misalkan kita ingin menggabungkan ordered list dengan unordered list.
